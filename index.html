// HOW IT WORKS

Divide: Split the array in half recursively until each subarray has 1 element (trivially sorted).

Conquer: Merge pairs of sorted subarrays. Use two pointers — one per half. Always pick the smaller front element. When one half is exhausted, append the rest of the other.

Why O(n log n)? There are log n levels of recursion (we halve each time). At each level, we do O(n) work in the merge step (every element is touched once). Total: n × log n.
